## 📋 نظرة عامة

**مشروع روافد** هو بنية تحتية مؤسسية للقطاع غير الربحي في المملكة العربية السعودية، تهدف إلى ربط أكثر من 5,000 جمعية أهلية في منظومة تكاملية متكاملة عبر 3 مكونات رئيسية:

| المكوّن | الهدف |
|---------|--------|
| 📚 الأدلة المرجعية المكيّفة | تحويل الأدلة الوطنية المركزية إلى أدلة قطاعية ومناطقية قابلة للتطبيق |
| 🔗 إطار التكامل القطاعي | تنظيم العلاقة بين الجمعيات وتوزيع الأدوار وتقليص التكرار |
| ⛓️ سلاسل القيمة التنموية | ربط الجمعيات عبر القطاعات لتقديم خدمة شاملة للمستفيد |

---

## 🚀 البدء السريع

```bash
# استنساخ المستودع
git clone https://github.com/your-org/rawafid.git
cd rawafid

# تثبيت التبعيات
npm install

# تشغيل بيئة التطوير
npm run dev
```

## 📦 المتطلبات

- **Node.js** >= 18.x
- **npm** >= 9.x أو **yarn** >= 1.22
- **Python** >= 3.10 (للخدمات التحليلية)
- **PostgreSQL** >= 14 (قاعدة البيانات)
- **Redis** >= 7.x (التخزين المؤقت)

---

## ⚙️ الإعداد والتكوين

1. انسخ ملف متغيرات البيئة:
   ```bash
   cp .env.example .env
   ```

2. عدّل `.env` بقيمك الخاصة:
   ```env
   DATABASE_URL=postgresql://user:pass@localhost:5432/rawafid
   REDIS_URL=redis://localhost:6379
   API_SECRET_KEY=your-secret-key
   ```

3. شغّل ترحيل قاعدة البيانات:
   ```bash
   npm run db:migrate
   ```
